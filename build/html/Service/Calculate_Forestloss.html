Calculate Forest Change#

Computing Forest Loss#

Note

The current release of the MISLAND-North Africa uses the High resolution Hansen Global forest Change data to compute forest loss for selected area and year.

To compute forest loss using the Hansen Global forest change dataset;

  1. On the top left conner of the Map pannel, click on the tool to toggle the drowing tools. Once the drawing tools are revealed, click on the tool to start drowing a custom area on the map where you wish to compute the forest loss

Draw a polygon tool#

Draw a polygon tool#

  1. Select and click on the layer settings icon . On the layer settings dialog, under the ‘SELECT DATA SOURCE’ options, choose ‘Hansen’ and select the year you wish to compute the forest loss.

Selecting the Hansen Forest loss data#
